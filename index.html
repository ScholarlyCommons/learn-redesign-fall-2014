This is a tab subtitle in a left column

If you want to insert columns into some tab content that is not set up for columns, it's easy to do!

When there is no column environment, the code looks like this:

<div class="row">
  <div class="col-sm-12">
    <h4>Tab Subtitle</h4>
    <p>Tab content ...</p>
  </div>
</div>

The layout class on the second div above means make this div a column that is twelve units wide, even on a small screen.

This layout style is a part of Bootstrap's grid layout, which is a library of style rules that help you control web page layout. The grid is based on rows that can be up to twelve units wide.

So if you want to present a two-column layout, that code will look something like this:

<div class="row">
  <div class="col-sm-6">
    <h4>Left column tab subtitle</h4>
    <p>Tab content ...</p>
  </div>
  <div class="col-sm-6">
    <h4>Right column tab subtitle</h4>
    <p>Tab content ...</p>
  </div>
</div>

Notice here how each of the two 6-unit columns are nested within the same row.

Want to make a 3-column layout? Easy!

Just do something like this:

<div class="row">
  <div class="col-sm-4">
    <h4>Left column tab subtitle</h4>
    <p>Tab content ...</p>
  </div>
  <div class="col-sm-4">
    <h4>Middle column tab subtitle</h4>
    <p>Tab content ...</p>
  </div>
  <div class="col-sm-4">
    <h4>Right column tab subtitle</h4>
    <p>Tab content ...</p>
  </div>
</div>

And this is a column on the right

Check out the text in the left column to learn how to create a two-column layout!

By the way...

The library that powers the web page layout is "responsive", which means that the layout changes if you look at it on a small device or when you make your browser smaller on your laptop or desktop. Responsive columns often get stacked on top of each other when viewed in smaller viewports. Resize your browswer window to see this in action!
